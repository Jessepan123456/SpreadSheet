# SpreadSheet
A recreation of the Google spreadsheet that supports data storage, editing, and formula evaluation

## Features
- Create and edit cell values
- Add, update, and delete cells
- Read from and write to files using JSON serialization
- Dynamic updates when cell values change
- Implement dependency tracking between cells

## Additional Features
- Created an Undo and Redo buttons using stacks
- Implemented a Clear button

## How to Run
1. Clone the repository
2. Run the program first
3. Run the GUI second for the spreadsheet
